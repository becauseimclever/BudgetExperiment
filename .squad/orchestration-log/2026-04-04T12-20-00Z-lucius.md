### Implementation Decisions

1. **DayIndex mapping** — `DayOfWeek.Sunday = 0` in C#; mapped to heatmap (Mon=0…Sun=6) via `dow == DayOfWeek.Sunday ? 6 : (int)dow - 1`. Used `DateOnly.DayNumber` for week-offset arithmetic.
2. **WeekIndex** — 0-based offset from earliest transaction's week: `(mondayOfThisWeek.DayNumber - mondayOfFirstWeek.DayNumber) / DaysInWeek`.
3. **Waterfall sort** — Sorted `CategorySpendingDto` by `Amount.Amount` ascending (all-negative amounts → descending by absolute value). Avoids `Math.Abs` overhead.
4. **Tukey's hinges** — Odd n: exclude median, split halves; even n: equal halves. `ComputeMedian` helper reused for both halves.
5. **Whiskers vs. outliers** — `Minimum`/`Maximum` are whisker endpoints; values outside Q1−1.5×IQR…Q3+1.5×IQR stored in `Outliers` collection.
6. **monthsBack reference** — `list.Max(t => t.Date)` from input dataset; makes tests deterministic regardless of wall-clock date.
7. **Candlestick grouping** — Sorted balances by date before `GroupBy` so `g.First()`/`g.Last()` are chronologically correct.
