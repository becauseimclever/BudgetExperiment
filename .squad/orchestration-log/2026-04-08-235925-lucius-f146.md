## Key Decisions

1. **ITransactionRepository.DeleteTransferAsync** returns `Task` (not `Task<bool>`).
   - Repo handles all 3 cases (none/orphan/both) silently.
   - Service wraps with existence check via `GetByTransferIdAsync` to return `bool`.

2. **Feature flag DELETE endpoint returns 403 Forbidden** (not 404) when disabled.
   - Signals the feature exists but is gated.

3. **:guid route constraint returns 404** (not 400) for non-GUID path segments.
   - This is ASP.NET Core routing behavior.

4. **When adding required constructor parameter** (e.g., `ILogger`), ALL test instantiation sites must be updated.
   - Use Python regex for bulk updates (PowerShell regex too slow on large files).
