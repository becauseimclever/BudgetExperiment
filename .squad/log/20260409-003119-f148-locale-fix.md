## Overview

Statement Reconciliation UI had 7 bare `.ToString("C")` calls across 4 Razor components, causing non-US users to see incorrect currency symbols (e.g., `¤` instead of `€` or `$`). Feature 148 fixed all instances and added 6 bUnit locale tests to prevent regression.

## Key Technical Decisions

1. **Inject Naming:** `@inject CultureService Culture` (not `CultureService CultureService`) — project convention for scoped/injected services.
2. **Extension Method:** `FormatCurrency()` from `CurrencyFormattingExtensions` — already globally available via `_Imports.razor`.
3. **Test Helper:** `TestCultureServiceFactory` — new shared pattern for setting up locale-aware bUnit tests.
4. **Service Stubs:** `StubBudgetApiService` extended with list properties for `ReconciliationHistory` and `ReconciliationTransactions`.
