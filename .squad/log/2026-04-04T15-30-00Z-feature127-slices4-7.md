## Patterns Established This Session

1. **SVG components:** Pure-Blazor `.razor`/`.razor.cs`/`.razor.css` triple-file pattern; `OnParametersSet()` precomputes geometry; `F()` for InvariantCulture doubles; `RenderSvgText()` MarkupString for SVG text elements.
2. **ApexCharts components:** Live in `src/.../Charts/ApexCharts/`; require `@namespace` directive; use `IServiceProvider.GetService<T>()` for optional theme/color services.
3. **SA1204:** `private static` before `private` instance — build fails otherwise. Applies to both methods and properties.
4. **bUnit + ApexCharts:** `JSRuntimeMode.Loose`; assert HTML wrapper only; never assert JS-rendered content.
