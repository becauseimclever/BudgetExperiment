## Feature Overview

Enable users to edit budget categories and goals directly from the calendar view. Users can tap/click on a calendar cell or overlay to adjust category allocations, set goals, or view progress for that period.

## Motivation

Direct calendar-based editing streamlines the budgeting workflow, making it more intuitive and time-centric. This reduces navigation friction and aligns with a calendar-first experience.

## Goals
- Edit category budgets/goals from calendar cells or overlays
- Visual feedback for changes (e.g., updated progress bars)
- Support for both mobile and desktop

## Acceptance Criteria
- Users can edit budgets/goals from the calendar
- Changes are reflected immediately in the calendar view
- UI is accessible and responsive
