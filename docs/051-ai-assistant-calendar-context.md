# AI Assistant Calendar Context

## Feature Overview

Ensure the AI Assistant uses the currently selected calendar date or range as context for all suggestions and actions. This makes AI-driven workflows more relevant and time-aware.

## Motivation

Contextual AI suggestions reduce manual input and errors, making the assistant more helpful and aligned with user intent.

## Goals
- AI Assistant suggestions reflect the active calendar selection
- Pre-fill or suggest transaction details based on selected date/range
- Consistent experience across mobile and desktop

## Acceptance Criteria
- AI Assistant suggestions are contextual to the selected calendar date/range
- Users can trigger AI actions from the calendar view
- UI is accessible and responsive
